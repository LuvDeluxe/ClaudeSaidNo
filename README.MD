# ClaudeSaidNo 🟢🔴

Tiny macOS menu bar app that tells you when Claude.ai peak hours are happening.

- **🟢 ClaudeSaidYes** = off-peak → go code heavy
- **🔴 ClaudeSaidNo** = peak (5–11 AM PT weekdays) → limits burn faster

Live countdown + your local time. No login. Zero bloat.

### Quick Start (easiest)

1. Download `claude_said_no.py`
2. Open Terminal and run:
   ```bash
   pip install rumps pytz tzlocal
   python claude_said_no.py
   ```
